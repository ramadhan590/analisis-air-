# Streamlit App - Air Quality Dashboard

## Menjalankan secara lokal
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy ke Streamlit Community Cloud
1. Upload `app.py` dan `requirements.txt` ke repository GitHub.
2. Buka Streamlit Community Cloud.
3. Pilih repository tersebut.
4. Set main file path ke `app.py`.
5. Klik Deploy.

## Catatan
- Data akan otomatis diunduh dari Google Drive saat app pertama kali dijalankan.
- App ini adalah versi dashboard dari notebook analisis Air Quality Dataset.
